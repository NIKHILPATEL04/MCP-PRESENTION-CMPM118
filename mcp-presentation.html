Definition

Model Context
Protocol

MCP is a protocol that allows AI models to interact with external tools and services through a standardized interface.

Instead of the AI directly calling APIs, the model communicates with an MCP server — which exposes system capabilities as structured, callable tools.

Core Concept

MCP acts as a bridge between AI reasoning and real software capabilities.

  • AI speaks MCP — not raw APIs
  • Tools are described, typed, and structured
  • Server handles execution safely
  • Same protocol works across all tools

Tools — 1 of 2

Exposed
Capabilities

The MCP server exposes five tools. Each connects directly to the internal engine. Here are the first two.

Map Inspection Tool

Allows the AI to read the current map state. The AI can request tiles in a region to understand the environment — retrieved directly from the database.

Tile Information Tool

Provides terrain type definitions and tile rules. The AI can ask what rules apply to specific tiles — forests, mountains, water — to reason about world structure.

Tools — 2 of 2

Remaining
Tools

Region Generation Tool

Triggers the map generation algorithm — calling pewter_generator.py which runs the layered prompting algorithm to produce valid terrain.

Map Validation Tool

Checks whether generated map violates spatial rules using pewter_rules.py: tile connectivity, adjacency constraints, overlap prevention.

Generation History Tool

Reads generation logs from the SQLite database. Allows the AI to inspect previous map states, generation decisions, and analytics for more informed reasoning.

Example Interaction

Forest Region
Generation

# Prompt to the AI agent
"Generate a forest region
 near this location."

This single instruction triggers a full MCP interaction cycle — the AI does not guess what terrain to generate, it acts on the real system.

What happens
  • AI reads map state via inspection tool
  • Selects the region generation tool
  • MCP server executes the generator
  • Layered prompting produces valid terrain
  • New tiles written to the database
  • Frontend map updates in real time

The AI actively interacts with the system — not just producing text responses.

Implementing an MCP server demonstrated clear structural and practical advantages over traditional AI-to-tool integration approaches.
